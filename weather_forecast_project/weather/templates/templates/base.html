Welcome to REST API Weather

Beforo starting, please, sing in. Type /authorize/username/password

To log out, please, type /logout

Type your city and units into url string to get weather info..

Format: http://127.0.0.1:8000/weather/latin_name_of_city/units (possible units: F, C, K)

Example: http://127.0.0.1:8000/weather/moscow/F
